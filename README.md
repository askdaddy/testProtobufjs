用ProtoBuf.js将JSON数据缓存转成二进制缓存
=====================================
edit by Seven on 2015-05-31

项目介绍
----------
1. 安装  `npm install`
2. 转换  `node main`
3. 测试  `node server`  

各程序功能
----------
1. main.js 用来将 hour.json 转成 www/hour.bin 。  
2. server.js 用node 启动后 访问 http://localhost:8080/ ,打开浏览器的调试看console
